# KPI Definitions

| KPI | Definition |
| --- | --- |
| MRR | Paid invoice revenue grouped by invoice month |
| ARR proxy | Sum of monthly recurring revenue across generated months |
| Active customers | Customers with active subscriptions |
| Trial customers | Customers with trial subscriptions |
| Churned customers | Cancelled subscriptions grouped by end month |
| Feature adoption | Events and active users grouped by feature |
| Customer health | Revenue and product usage minus support burden |
| Quality score | One minus validation issue rate |
